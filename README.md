## module-app
WanAndroid练手项目

## 为什么选择Kotlin
>* 简洁：大大减少样板代码的数量
>* 安全：避免空指针异常等整个类的错误
>* 互操作性：充分利用JVM，Android和现有浏览器的库；
>* 工具友好：可用任何Java IDE 或者使用命令行构建
